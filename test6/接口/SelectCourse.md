# 接口：SelectCourse  [返回](../README.md)
用例： [选课](../用例/选课.md)
- 功能：
    用户选择自己的课程。
    
- 权限：
    学生/老师已经登录。    
    
- API请求地址： 
    接口基本地址/v1/api/SelectCourse

- 请求方式 ：
    POST

- 请求实例：

        {
            "use_id":"201510414426",
			"course_id":"16479"
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
|user_id|已经登录用户的user_id值，对应表USERS.USER_ID的值|
|course_id|课程名|
  
- 返回实例：

        {         
            "status": true,
            "info": null,    

        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
